# IT-Boost | Современный интерактивный лендинг с киберпанк-вайбом

Адаптивный промо-сайт (лендинг) для IT-компании или Digital-агентства. Проект выполнен в ярком стиле маджента/киберпанк и содержит продвинутые фронтенд-фишки для максимального вовлечения пользователей.

## 🛠 Технологический стек
* **Разметка и структура:** HTML5 (Семантическая верстка).
* **Стилизация:** CSS3 (Flexbox, Grid, Glassmorphic эффекты).
* **Интерактив и логика:** JavaScript (ES6+).
* **Анимации:** AOS (Animate On Scroll) библиотека.
* **Бэкенд формы:** FormSubmit API (Облачный бэкенд без перезагрузки страницы).

---

## 🚀 Что реализовано в проекте (Фичи)

1. **Динамическая смена темы (Dark/Light):** Реализован полноценный переключатель тем (Солнце/Луна). Логика завязана на CSS-переменные (`--variables`) и JS, что позволяет сайту плавно менять цвета без перезагрузки.
2. **Эффект Glassmorphism & Параллакс:** Шапка сайта и карточки услуг выполнены в трендовом стиле «матового стекла» (`backdrop-filter: blur`). На главном экране настроен параллакс-эффект для фонового изображения.
3. **Кастомный интерактивный курсор:** Стандартная стрелка Windows скрыта, вместо нее разработан кастомный круглый курсор с плавным шлейфом (`transition`), реагирующий на движения мыши.
4. **Анимация появления при скролле (AOS):** Элементы интерфейса, карточки преимуществ и форма обратной связи плавно "выплывают" с разной задержкой (`delay`) по мере прокрутки страницы.
5. **Интерактивное мобильное меню (Бургер):** Полностью адаптивная навигация. На мобильных устройствах меню скрывается в бургер-иконку и плавно открывается по клику.
6. **Рабочая форма без своего бэкенда:** Интегрирован сервис FormSubmit. Форма полностью валидна и отправляет реальные заявки на почту.
